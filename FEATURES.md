# Features

## Implemented

- Modular Next.js application scaffold.
- Authentication UI pages and workflow.
- Dashboard layout and reusable UI primitives.
- Basic CRM workspace with forms for leads, companies, deals, tasks, and notes.
- Organization pages skeleton with membership and settings flows.
- Supabase client wrappers for browser and server environments.
- Project documentation and sprint planning artifacts.

## Planned

- Production-grade tenant management with org-scoped auth.
- RBAC permission enforcement for workspace access.
- Full Supabase schema alignment and row-level security.
- Lead discovery integration with Google Places.
- Business import, duplicate detection, and refresh workflows.
- Analytics and AI-driven reporting.
- Marketing automation and campaign management.
