# User Stories

## User Story 001

- As a Marketing Manager
- I want to search for local businesses in Delhi by keyword, industry, and radius
- So that I can build a targeted outreach list and import businesses into my organization.

### Acceptance Criteria
- Authenticated users can access lead discovery.
- Search supports keyword, industry, radius, city, state, and country.
- Results are paginated and exportable.
- Duplicate business imports are detected and prevented.
- Data is isolated by organization.
