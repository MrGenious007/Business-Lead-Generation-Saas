# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in LeadPilot AI, please report it responsibly by opening an issue or contacting the maintainers directly.

Do not publish the vulnerability publicly until it has been addressed.

## Supported Versions

This repository is actively under development. Security support is currently provided for the latest active branch.

## Security Best Practices

- Do not commit secrets, API keys, or credentials to the repository.
- Store environment variables securely in deployment settings.
- Use server-side API routes for sensitive credentials and operations.
- Validate all inputs on the server side in addition to client-side checks.

## Response

We will review security reports promptly and respond with remediation steps as soon as possible.
