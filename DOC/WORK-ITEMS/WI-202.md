## Objective

Establish a complete, type-safe, and validated foundation for the CRM module by defining TypeScript models, Zod schemas, DTOs, and shared constants that align with the production database schema.

## Scope

- Update CRM entity TypeScript models in `types/crm.ts` to match `011_crm_database_foundation.sql`.

- Integrate validators into `components/crm/crm-forms.tsx` using React Hook Form.
- Update `services/crm.ts` to use the new types and inject the active organization on create operations.
- Add Jest validation tests in `tests/crm.test.ts`.
